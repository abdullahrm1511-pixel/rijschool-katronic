# Rijschool Katronic

Schone iPhone-only SwiftUI app voor Xcode.

## Openen op Mac

1. Clone deze repository.
2. Open `RijschoolKatronic.xcodeproj` in Xcode.
3. Sluit een iPhone aan.
4. Kies je Apple ID / Team bij `Signing & Capabilities`.
5. Druk op Run.

## Inhoud

- `RijschoolKatronic.xcodeproj`: Xcode-project
- `RijschoolKatronic/`: SwiftUI broncode
- `LEESMIJ_MAC_XCODE.md`: korte installatiehulp

De app gebruikt lokale opslag op de iPhone.
